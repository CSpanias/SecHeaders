HSTS demo

This HTTPS host can send the Strict-Transport-Security header. When your browser receives the header it will upgrade future http:// requests to https:// for the host.

  • Visit (HTTPS): https://example.test/
  • Then visit (HTTP): http://example.test/ — the browser should automatically use HTTPS instead (URL will change or connection will be upgraded).

Use curl -I https://example.test/ to inspect headers (curl won’t enforce HSTS; browsers do).
